#### My Solution

简单模拟，通过100%

```python
# MiHoYo Q1
from math import ceil

n, m = map(int, input().split())
# n, m = 3200, 35
month_card = ceil(n / 3000)
ans = ceil(n / 10)
for i in range(month_card + 1):
    money = i * 30
    need_stones = n - 300 * i
    need_stones -= 90 * min(m, i * 30)

    if need_stones <= 0:
        ans = min(ans, money)
        continue
    money += ceil(need_stones / 10)
    ans = min(ans, money)
print(ans)
```
